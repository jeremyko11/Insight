## 🧩 什么是"人性原子"？

原子就是**可独立复用的人性洞见乐高**，每条原子格式：

```json
{
  "id": "2026_type_001",
  "knowledge": "一句话反常识、能戳人的洞见",
  "original_source": "出处",
  "topics": ["婚姻", "心理", "职场"],
  "type": "anti-pattern", // principle / viral-insight / case
  "confidence": "high"
}
```

**type 说明：**
- `principle`：公理级人性原则
- `anti-pattern`：反常识破局（戳破常见误区）
- `viral-insight`：病毒性洞见（适合做标题）
- `case`：具体案例总结

**id 编号规则：**
```
年份_类型缩写_序号
例如：2026_viral_001 → 2026年 第1条 viral-insight
```

**可用 topics：**
`婚姻,心理,社会心理,社会观察,人性暗黑,情感,热点,两性,原生家庭,职场,职场人性`

## 🚀 使用方法（OpenClaw环境）

### 方式一：全自动流程（推荐）

直接输入：
```
/insight 你的话题
```
例如：
```
/insight 为什么现代人越来越焦虑
/insight 不敢休息，一躺下这个家就运转不了
/insight 为什么越听话的员工走得越早
```

**自动流程：**
```
用户输入
  ↓
insight 主入口识别意图
  ↓
insight-diagnosis → 消解浅层假设，输出三个反常识角度
  ↓
topic-finder → 深挖三个热点角度，预测热度
  ↓
copy-optimizer → 输出微博/小红书/抖音三平台优化文案
  ↓
完成！你直接复制就能发
```

### 方式二：单独使用某个模块

如果你只需要某个环节：

```
/insight-diagnosis 你的话题     # 只做消解诊断，输出三个角度
/topic-finder 你的话题         # 只挖热点角度
/copy-optimizer 你的原文案     # 只优化文案
/ip-builder 帮我包装XX这个IP  # 只输出IP包装方案
```

## 📝 如何补充原子库？

原子库越大，工具越聪明。按这个步骤补充：

### Step 1：收集素材
- 关注5-10个头部情感/人性/职场博主，摘他们点赞高的洞见
- 从心理学书籍摘经典结论
- 自己的生活观察，一句话总结

### Step 2：批量提炼
1. 打开 `批量原子提炼Prompt.md`，完整复制内容
2. 粘贴给AI（Claude/Grok/Kimi都可以）
3. 在后面贴上你收集的原文
4. AI输出JSON数组

### Step 3：追加到 atoms.jsonl
- 把每个JSON对象拆开，**一行一个**
- 追加到 `atoms.jsonl` 文件末尾
- 保持id递增

**示例正确格式：**
```json
{"id": "2026_viral_001", "knowledge": "...", "original_source": "...", "topics": ["..."], "type": "viral-insight", "confidence": "high"}
{"id": "2026_viral_002", "knowledge": "...", "original_source": "...", "topics": ["..."], "type": "viral-insight", "confidence": "high"}
```

### Step 4：（可选）验证JSON语法
```powershell
Get-Content atoms.jsonl | ForEach-Object { $_ | ConvertFrom-Json | Out-Null }; Write-Host "All OK"
```
如果没有报错，说明语法正确。

## 🎨 输出格式说明

### copy-optimizer 会输出三个平台的版本：

| 平台 | 长度 | 特点 |
|---|---|---|
| **微博** | 200-300字 | 直接戳，快节奏，结尾留讨论 |
| **小红书** | 300-500字 | 多分小段，有标签，结尾引导评论 |
| **抖音** | 15-30秒口播 | 开头3秒炸，口语化，结尾提问 |

每个版本都自动加上：
- 你的IP签名：`—— @你的ID · 人性观察第X期`
- 适合的标签（小红书）
