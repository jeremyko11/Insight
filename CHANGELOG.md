## 1.0.0 - 2026-03-23

### Features
- Initial release of insight-skill:人性观察 + IP内容引擎
- Fully automatic workflow: `input → diagnosis → hotness → copy → feedback → update atoms`
- Five modules:
  - `insight`: main entry, full automatic pipeline
  - `insight-diagnosis`: digest shallow assumptions, output three anti-common-sense angles
  - `topic-finder`: find hot potential angles, predict virality
  - `copy-optimizer`: output optimized copy for Weibo/Xiaohongshu/Douyin
  - `ip-builder`: personal IP positioning and packaging
- Core 7 axioms for human nature observation
- Atomic insight library with JSONL format for continuous evolution
- Content evolution system: viral content → re-atomize → update library → get smarter over time
- All required modules included, one-step installation for OpenClaw
- Anti-abuse mechanism: avoid confrontation, avoid over-simplification
