## 1.0.0 - 2026-03-23

### Features
- Initial release of /insight insight content creation toolkit
- Single unified entry: `/insight topic`
- Complete workflow: topic → diagnosis → full article → cognitive model → IP quotes
- Support for multi-topic analysis: marriage, human nature, workplace, society, psychology
- Based on dontbesilent business diagnosis framework
- **v1.0 Core Upgrade**:
  - Double-layer cognitive structure: viral content + structural cognition
  - New formula: `popular content = emotion trigger × cognitive dislocation × reusable structure`
  - Three-layer human nature model: `behavior = bio-drive × game-theory × narrative self-consistency`
  - Content evolution closed-loop: input → digest → structure → generate → feedback → update atom library
  - Fixed output template: hook → explanation → scenario → conclusion → interaction
  - Anti-abuse mechanism: avoid confrontation, avoid over-simplification
- All required skills included, one-step installation
