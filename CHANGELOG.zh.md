## 1.0.0 - 2026-03-23

### 新功能
- 首次发布 /insight 热点洞察内容创作工具
- 统一入口：`/insight 话题` 一键触发
- 完整工作流：话题 → 诊断 → 完整文案 → 认知模型 → IP 金句
- 支持多话题分析：婚姻、人性、职场、社会、心理等各类话题
- 基于 dontbesilent 商业诊断框架
- **v1.0 核心升级**：
  - 双层认知结构：病毒内容（流量） + 结构认知（信任沉淀）
  - 新爆款公式：`爆款内容 = 情绪触发 × 认知错位 × 可复用结构`
  - 人性拆解三层模型：`行为 = 生物驱动 × 关系博弈 × 叙事自洽`
  - 内容进化闭环：输入 → 消解 → 结构化 → 生成 → 反馈 → 更新原子库
  - 固定输出模板：钩子 → 解释 → 场景 → 结论 → 互动
  - 反滥用机制：避免制造对立、避免过度简化
- 所有技能完整包含，一步安装，开箱即用
