# topic-finder：热点话题挖掘

从一个普通话题中，挖掘出**3个能上热点的反常识角度**。遵循七大公理，永远找"反着说"的方向。

---

## 挖掘原则

### 热点三要素（必须满足）
1. **反常识**：和大众普遍认知相反，产生认知冲突
2. **情绪钩子**：戳中某群人被压抑的情绪
3. **可讨论**：有争议空间，能引发评论区吵架

### 挖掘公式
```
大众说A → 你说not A → 为什么not A → 用人性本质解释
```

### 常见切入方向
- 从"生物本能"看问题 → 反对"社会规训"
- 从"自私基因"看问题 → 反对"无私道德"
- 从"当事人利益"看问题 → 反对"旁观者正确"
- 从"长期合同"看婚姻 → 反对"爱情神话"

---

## 工作流程

### Step 1：现状总结
一句话概括：大众普遍怎么看这个话题？

### Step 2：三角度挖掘
按照"高热度 → 中热度 → 深度角度"顺序输出：
1. 最炸：情绪最激烈，最容易上热门
2. 次炸：有争议，但相对温和
3. 深度：更本质，适合深度笔记

### Step 3：热度预测
给每个角度标注热度预测（⭐数量），并说明为什么这个角度能火。
