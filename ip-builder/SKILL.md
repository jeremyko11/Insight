# ip-builder：IP包装输出

打造**"真话有毒"的人性观察家人设**。核心是：让别人看到自己不敢承认的部分。

---

## IP定位公式

```
人设 = 立场 + 风格 + 固定句式 + 持续输出
```

### 立场（必须明确）
你是**"戳破皇帝新衣的孩子"**：
- 不说政治正确的废话
- 只说别人不敢说的人性真相
- 站在"被压抑情绪"那一边，不站在道德高地

### 风格（两种选择，选一种）
1. **毒舌犀利型**：直接戳破，不留情面，"我就是敢说真话"
2. **温和洞察型**：温柔而坚定，点破不说破，"我懂你的不容易"

### 固定标识（让人记住你）
- 固定开篇："今天这句话，可能会得罪很多人，但我还是要说"
- 固定结尾："—— @你的ID · 人性观察第X期"
- 固定标签：#人性 #心理学 #婚姻 #情感

---

## 工作流程

### Step 1：定位诊断
- 你的现状：现在内容是什么风格？
- 你的优势：你最擅长聊什么话题？
- 差异化：你和其他情感/心理博主有什么不一样？

### Step 2：输出包装方案
- 推荐人设定位
- 简介怎么写
- 内容固定结构
- 互动话术模板

### Step 3：长期成长路径
告诉你每个阶段该做什么，怎么从0到1放大IP。

## 内容固定结构（每次都这么写）

```
【标题】反常识提问/结论

【开头】一句话钩子（戳中情绪）
【中间】3个小点，每点一句话洞见 + 简短解释
【结尾】总结 + 引导评论 + IP签名
```

---

## 互动话术模板

### 引导评论
- "说说你的经历，我看看有多少人懂？"
- "你遇到过这种情况吗？评论区聊聊。"
- "同意的点个赞，不同意评论区告诉我。"
- "你的看法是什么？欢迎留下你的观点。"

### 固定结尾签名
```
—— @正直清泉 · 人性观察第X期
```

## 长期IP成长路径

### 阶段一：冷启动（0-1万粉）
- 每天1条，固定时间输出
- 只聊一个垂直领域（婚姻/人性/情感选一个）
- 每条都带反常识，每条都引导评论
- 目标：让系统知道你是谁，给你打标签

### 阶段二：放大（1-5万粉）
- 保持更新频率，开始做合集
- 引导粉丝加微信，沉淀私域
- 开直播聊粉丝问题，增强粘性
- 目标：建立稳定流量基本盘

### 阶段三：变现（5万粉+）
- 1对1咨询（199元/小时）
- 线上课程（99元/系列）
- 年度社群（365元/年）
- 广告合作
