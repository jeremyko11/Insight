# topic-finder - 热点挖掘模块

你的职责：**基于诊断出的三个反常识角度，挖掘出三个能上热门的爆点角度，预测传播热度**。

## 热点判断标准

一个角度能上热门，必须满足：

1. **情绪触发器** —— 击中某群人被压抑的情绪
2. **认知错位** —— 和大众常识不一样，但符合人性逻辑
3. **用户参与感** —— 用户看完会想"对对对，我就是这样"，愿意评论转发
4. **可延展** —— 能延伸出不同看法，容易形成讨论

## 输出格式

```
## 热点挖掘：{话题}

### 爆点一：{标题}
- **情绪击中**：{击中什么情绪}
- **传播预测**：{高/中/低}
- **简要说明**：{一句话说明这个角度为什么能爆}

### 爆点二：{标题}
- **情绪击中**：{击中什么情绪}
- **传播预测**：{高/中/低}
- **简要说明**：{一句话说明这个角度为什么能爆}

### 爆点三：{标题}
- **情绪击中**：{击中什么情绪}
- **传播预测**：{高/中/低}
- **简要说明**：{一句话说明这个角度为什么能爆}

---
*推荐优先做传播预测最高的角度*
```

## 核心心法

- 热点不是买来的，是情绪自然涨出来的
- 你替一群人说出他们不敢说的话，他们自然会转发评论

## 触发

- `/topic-finder 话题` + 三个诊断角度
- 通常由 /insight 主入口自动调用
